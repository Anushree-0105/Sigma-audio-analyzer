# Call Analyzer Backend

This repository contains the backend and frontend for Call Analyzer.

Setup (local):

1. Install dependencies for backend (Python/Node as needed).
2. Install frontend deps in `call-analyzer-frontend` and run with your preferred tool (e.g., `npm install && npm run dev`).

To push to a remote repository (after creating a remote on GitHub/GitLab):

```bash
git remote add origin <REMOTE_URL>
git branch -M main
git push -u origin main
```
